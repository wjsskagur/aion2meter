# Aion2Meter

아이온2 실시간 DPS 미터기

파티원의 데미지 순위, 보스 HP, 전투 시간을 게임 화면 위에 반투명으로 표시합니다.

---

## 주요 기능

- 파티원 데미지 순위 및 비율 실시간 표시
- 보스 HP 바 및 남은 HP 수치
- 전투 시간 타이머
- 전투 기록 (최근 20회)
- 플레이어 스킬별 상세 통계
- 항상 위 / 투명도 / 폰트 크기 조절
- 전투 자동 종료 감지

---

## 설치 방법

**1. 다운로드**

[최신 버전 다운로드](https://github.com/wjsskagur/aion2meter/releases/latest)에서 `Aion2Meter-Setup.exe`를 받습니다.

**2. 설치**

`Aion2Meter-Setup.exe`를 실행하면 Npcap(패킷 캡처 드라이버)이 자동으로 함께 설치됩니다.

**3. 실행**

바탕화면 바로가기 또는 시작 메뉴에서 **Aion2Meter**를 실행합니다.
관리자 권한을 요청하면 허용해 주세요.

---

## 사용 방법

1. 아이온2를 먼저 실행하고 게임에 접속합니다.
2. Aion2Meter를 실행합니다. 상태 표시줄에 **캡처 중...** 이 표시되면 준비 완료입니다.
3. 던전에 입장해서 보스를 공격하면 자동으로 데미지가 집계됩니다.
4. 전투 종료 후 일정 시간(기본 10초) 공격이 없으면 자동으로 기록이 저장됩니다.

### 창 조작

| 동작 | 방법 |
|------|------|
| 창 이동 | 상단 제목 영역을 드래그 |
| 창 크기 조절 | 우측 하단 모서리를 드래그 |
| 데미지 초기화 | ↺ 버튼 클릭 |
| 전투 기록 보기 | 📋 버튼 클릭 |
| 설정 열기 | ⚙ 버튼 클릭 |

### 설정

| 항목 | 설명 |
|------|------|
| 항상 위 | 게임 화면 위에 항상 표시 |
| 투명도 | 창 투명도 조절 (0.3 ~ 1.0) |
| 폰트 크기 | 텍스트 크기 조절 |
| 컴팩트 모드 | 행 높이를 줄여 좁게 표시 |
| 알려진 플레이어만 표시 | 닉네임이 확인된 파티원만 표시 |
| 내 캐릭터 상단 고정 | 내 딜량을 항상 맨 위에 표시 |
| 자동 종료 대기 시간 | 전투 종료 감지까지 대기할 초 |
| 정렬 기준 | 총 데미지 / DPS / 타격 수 |

## 자주 묻는 질문

**Q. 데미지가 표시되지 않아요.**
- Aion2Meter를 관리자 권한으로 실행했는지 확인하세요.
- 게임을 먼저 실행한 후 Aion2Meter를 켜주세요.
- 상태 표시줄에 오류 메시지가 있다면 ↺ 버튼을 눌러 재시작해 보세요.

**Q. 닉네임이 직업명(검성, 수호성...)으로 표시돼요.**
- 입장 시 닉네임 패킷을 놓친 경우입니다. 해당 플레이어가 사망 후 부활하거나 재접속하면 자동으로 업데이트됩니다.

**Q. 보스 HP가 표시되지 않아요.**
- 일부 보스는 HP 패킷을 보내지 않을 수 있습니다. 데미지는 정상적으로 집계됩니다.

**Q. 백신이 차단해요.**
- 네트워크 패킷을 캡처하는 특성상 일부 백신이 오탐할 수 있습니다. 예외 처리 후 사용해 주세요.

**Q. 게임 내 처벌 위험이 있나요?**
- Aion2Meter는 네트워크 패킷을 읽기만 하며 게임 클라이언트나 서버에 어떠한 수정도 가하지 않습니다. 단, 모든 서드파티 툴 사용에 대한 책임은 사용자에게 있습니다.

---

## 주의사항

- 게임 업데이트로 패킷 구조가 변경되면 동작하지 않을 수 있습니다.
- 관리자 권한 없이는 패킷 캡처가 불가능합니다.
- 게임사 공식 도구가 아닙니다.
